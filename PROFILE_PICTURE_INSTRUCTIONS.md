# How to Add Your Profile Picture

## 🖼️ Step-by-Step Instructions

### 1. **Save Your Image**
- Take your profile picture (the one with you in front of the temple)
- Save it as `me.jpg` in the same folder as `portfolio.html`
- Make sure the filename is exactly: `me.jpg` (all lowercase)

### 2. **Image Requirements**
- **Format**: JPG, JPEG, PNG, or WebP
- **Size**: At least 400x400 pixels (larger is better)
- **Aspect Ratio**: Square (1:1) works best
- **File Size**: Under 5MB for faster loading

### 3. **File Location**
Your folder should look like this:
```
portfolio/
├── portfolio.html
├── me.jpg  ← Your image here
├── README.md
└── PROFILE_PICTURE_INSTRUCTIONS.md
```

### 4. **Alternative Formats**
If your image is in a different format, you can:
- **PNG**: Save as `me.png` and update the HTML
- **WebP**: Save as `me.webp` and update the HTML

### 5. **If Image Still Doesn't Show**
The portfolio now has a fallback system:
- If the image fails to load, it will show a user icon instead
- This ensures your portfolio always looks good

### 6. **Testing**
1. Save your image as `me.jpg`
2. Open `portfolio.html` in your web browser
3. You should see your profile picture in the circular frame
4. If not, check that the filename is exactly correct

## 🔧 Troubleshooting

### Image Not Showing?
- Check the filename is exactly `me.jpg`
- Make sure the file is in the same folder as `portfolio.html`
- Try refreshing the browser (Ctrl+F5 or Cmd+Shift+R)

### Image Looks Distorted?
- Use a square image (1:1 aspect ratio)
- Make sure the image is at least 400x400 pixels
- The image will automatically be cropped to fit the circular frame

### File Too Large?
- Compress your image to under 5MB
- Use online tools like TinyPNG or Squoosh
- Convert to WebP format for smaller file size
